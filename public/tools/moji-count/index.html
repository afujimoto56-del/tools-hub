用途別 文字数カウント

文章を貼ると、文字数をその場で数えます。X(Twitter)や原稿用紙、メタディスクリプションなど、用途ごとの上限で「あと何文字か」が分かります。

0 文字
空白なし 0 文字
用途上限残り

何を数えている？

「文字数」は改行や空白も含めた全体の長さ、「空白なし」は半角・全角スペースと改行を除いた数です。原稿用紙は400字＝1枚として枚数を表示します。

用途別の上限について

X(Twitter)の投稿は全角140字、検索結果に出るメタディスクリプションは概ね120字前後が目安です。各用途で「上限 − 現在の文字数」を残りとして表示します。残りがマイナスなら超過しています。
